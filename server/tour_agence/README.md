<h3 style="text-align:center">To run the project follow the following commands</h3>
    
         1. composer install
         2. composer upgrade 
         3. composer update
         4. cp .env.example .env
         5. php artisan config:clear
         6. php artisan serve
         7. npm install && npm run dev
